# Gerber Files

Place the Gerber manufacturing export from EasyEDA here before ordering PCBs.

## How to Export from EasyEDA

1. Open the PCB design in EasyEDA
2. Fabrication → Generate Gerber
3. Download the ZIP file
4. Extract and place files in this folder

## Required Layers

| File Extension | Layer |
|----------------|-------|
| `.GTL` | Top copper |
| `.GBL` | Bottom copper |
| `.GTS` | Top solder mask |
| `.GBS` | Bottom solder mask |
| `.GTO` | Top silkscreen |
| `.GBO` | Bottom silkscreen |
| `.TXT` / `.DRL` | Drill file |
| `.GKO` | Board outline |

## Ordering

Upload the ZIP directly to:
- [JLCPCB](https://jlcpcb.com) — cheapest, ~€5 for 5 boards
- [PCBWay](https://pcbway.com) — good quality, slightly more expensive
- [Aisler](https://aisler.net) — EU-based, faster shipping to Europe
