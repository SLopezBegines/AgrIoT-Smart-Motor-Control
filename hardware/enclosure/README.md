# Enclosure

Currently using an off-the-shelf IP65 waterproof enclosure (200×120×75 mm).

A custom 3D-printable enclosure for the control node PCB is planned for a
future version. Requirements:

- IP54 or better (outdoor/agricultural environment)
- DIN rail mount option
- Cable entry points for power (12 V), relay outputs, and sensor runs
- Ventilation slots with mesh (engine bay can reach 60°C)
- Clear lid option for LED status visibility

Place STL/STEP files here when available.
